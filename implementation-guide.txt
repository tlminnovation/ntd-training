SKINAPP TRAINING COURSE - H5P IMPLEMENTATION GUIDE
====================================================

OVERVIEW
--------
This H5P training course is designed to prepare medical staff for the WHO SkinNTD AI Image Collection Initiative. The course consists of 5 modules plus a final assessment.

COURSE STRUCTURE
----------------
Module 1: Project Overview & Objectives
- Introduction to the project
- Understanding the AI training goals
- Governance structure
- Your role as a clinician
- Format: Course Presentation with knowledge check

Module 2: Clinical Photography Best Practices
- Image quality requirements
- Required metadata fields
- Photography do's and don'ts
- Special considerations for private areas
- Format: Course Presentation with interactive elements

Module 3: Informed Consent Procedures
- Real-world scenarios
- Language barriers
- Ongoing consent
- Withdrawal of consent
- Format: Branching Scenario (interactive decision-making)

Module 4: Anonymization & Privacy Protection
- Why anonymization matters
- What to remove from images
- Technical safeguards
- Before/after examples
- Format: Course Presentation with examples

Module 5: Safeguarding Principles
- Five principles of dignity
- Vulnerable populations
- Red flags to watch for
- Mandatory reporting
- Format: Course Presentation with scenario examples

Final Assessment
- 20 comprehensive questions
- Covers all 5 modules
- 80% pass rate required
- Multiple question types (MCQ, True/False)

HOW TO USE THESE FILES
----------------------

OPTION 1: Moodle LMS (Recommended for WHO)
1. Install H5P plugin in Moodle
2. Navigate to your course
3. Add "H5P" activity
4. Upload each .h5p.json file
5. Configure completion tracking
6. Link modules sequentially

OPTION 2: H5P.com (Cloud-based)
1. Create account at h5p.com
2. Use "Import" function
3. Upload each JSON file
4. Organize into course structure
5. Share via link or embed

OPTION 3: WordPress with H5P Plugin
1. Install H5P plugin
2. Create new H5P content
3. Upload JSON files
4. Embed in pages/posts
5. Use LMS plugin for tracking

OPTION 4: Drupal with H5P Module
1. Enable H5P module
2. Create H5P content type
3. Import JSON files
4. Configure permissions
5. Track via Drupal's quiz module
